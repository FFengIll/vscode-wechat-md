## Overview

A VSCode extension (freshly scaffolded) designed to render Markdown in a WeChat Official Account (微信公众号) compatible format, enabling one-click copy-paste from editor to the WeChat platform.

## Current State

Bare scaffold — only a `helloWorld` command exists. No feature implementation yet.

## Key Files

- `package.json` — extension manifest, commands, activation events
- `tsconfig.json` — TS config (Node16 modules, ES2022, strict)
- `src/extension.ts` — main extension logic (currently helloWorld only)

## Integration Points

- VSCode Webview API — for rendering HTML preview
- VSCode clipboard API — for copy-to-clipboard
- VSCode commands — registered in `package.json` contributes.commands

## Patterns

- Standard VSCode extension pattern: `activate()` registers commands via `vscode.commands.registerCommand`
- Context subscriptions pattern for resource cleanup

## What Needs to Be Built

1. Markdown parser (marked / markdown-it)
2. WeChat-compatible CSS styles (inline styles, no external CSS)
3. WebviewPanel for preview
