# 微信公众号排版演示

这是一篇用于演示 **vscode-wechat-md** 扩展渲染效果的示例文档。打开预览面板，即可看到所有元素的微信公众号样式效果。

---

## 目录

本文涵盖以下排版元素：段落、标题、列表、代码、引用、表格、图片、链接、强调、分割线。

## 段落与强调

普通段落文字会按照微信公众号的阅读习惯进行排版，行高舒适，字号适中，适合在手机端阅读。

可以在段落中使用 **粗体** 突出重点，使用 *斜体* 表示术语或书名，也可以同时使用 ***粗斜体*** 来加强语气。

---

## 列表

### 无序列表

- 实时预览：编辑 Markdown 时右侧同步渲染
- 一键复制：富文本直接粘贴进公众号编辑器
- 自定义主题：通过 CSS 变量调整配色和字号
  - 修改主色调
  - 调整字体大小
  - 设置行高

### 有序列表

1. 打开任意 `.md` 文件
2. 点击编辑器标题栏的预览图标
3. 在预览面板中点击 **✂️ 复制内容**
4. 打开微信公众号编辑器，直接粘贴

---

## 代码

### 行内代码

使用 `Cmd+Shift+P` 打开命令面板，执行 `WeChat MD: Preview Markdown` 命令。

主题文件路径为 `.wechat/theme.css`，可通过修改 `--wechat-accent` 变量调整主色调。

### 代码块

```javascript
// 自定义主题变量示例
const theme = {
  accent: '#07C160',      // 微信绿
  fontSize: '16px',
  lineHeight: 1.8,
  textColor: '#333',
};
```

```css
/* .wechat/theme.css */
:root {
  --wechat-accent: #07C160;
  --wechat-font-size: 16px;
  --wechat-line-height: 1.8;
  --wechat-text-color: #333;
  --wechat-code-bg: #f6f8fa;
  --wechat-inline-code-color: #d63384;
  --wechat-blockquote-bg: #f9f9f9;
  --wechat-max-width: 680px;
}
```

```python
# Python 示例
def render_markdown(text: str) -> str:
    """将 Markdown 渲染为微信公众号兼容的 HTML"""
    renderer = WeChatRenderer()
    return renderer.render(text)
```

## 引用块

> 微信公众号编辑器不支持外部 CSS，因此所有样式必须以内联方式注入到每个 HTML 元素上。本扩展自动完成这一过程，确保粘贴后格式完整保留。

> **提示：** 如果图片无法显示，请使用 **✂️ 复制内容** 而非 **📋 复制 HTML**。前者会将本地图片自动转换为 Base64 编码。

## 表格

以下是扩展支持的 Markdown 元素汇总：

| 元素     | 示例语法       | 渲染效果       |
| -------- | -------------- | -------------- |
| 粗体     | `**文字**`     | **文字**       |
| 斜体     | `*文字*`       | *文字*         |
| 行内代码 | `` `code` ``   | `code`         |
| 链接     | `[文字](url)`  | 蓝色可点击链接 |
| 图片     | `![alt](path)` | 居中图片       |
| 引用     | `> 文字`       | 左边框引用样式 |
