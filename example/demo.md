# VSCode Markdown到微信公众号

公众号排版，向为文人之苦。

Markdown 既就，尚需转入编辑器，逐段调格、手动上图，繁琐耗时，所得未必如意。

余自制一工具，名曰 **wechat-md**，于 VS Code 中写罢，一键复制，粘贴即发，格式毫厘不失。

---

## 这是什么

![](demo.png)

**wechat-md（vscode-wechat-md）** 是一个 VS Code 扩展，专为微信公众号写作设计：

- **实时预览**：右侧面板同步渲染微信公众号样式，所见即所得
- **一键复制**：点击按钮，带完整内联样式的富文本写入剪贴板，直接粘贴进公众号编辑器
- **本地图片自动处理**：复制时自动转为 Base64，无需手动上传素材库
- **自定义主题**：通过 CSS 变量调整配色、字号、行高，一次配置长期使用
- **完全本地化**：本地运行，无需登录，无需授权

> 仓库：https://github.com/FFengIll/vscode-wechat-md
>
> 插件：https://marketplace.visualstudio.com/items?itemName=FFengIll.vscode-wechat-md
> 
> VSCode：扩展市场搜索 `wechat-md` 即可安装

---

## 使用方法

1. 安装扩展后，打开任意 `.md` 文件
2. 点击编辑器右上角的预览图标，打开实时预览面板
3. 写完文章后，点击顶部工具栏 **✂️ 复制内容**
4. 打开微信公众号编辑器，`Ctrl+V` / `Cmd+V` 粘贴，完成

## 段落与强调

普通段落文字会按照微信公众号的阅读习惯进行排版，行高舒适，字号适中，适合在手机端阅读。

可以在段落中使用 **粗体** 突出重点，使用 *斜体* 表示术语或书名，也可以同时使用 ***粗斜体*** 来加强语气。

## 代码

### 行内代码

使用 `Cmd+Shift+P` 打开命令面板，执行 `WeChat MD: Preview Markdown` 命令。

主题文件路径为 `.wechat/theme.css`，可通过修改 `--wechat-accent` 变量调整主色调。

### 代码块

> 预览代码块有一些差别，但复制粘贴后会适配到微信公众号相同的样式。
> 公众号编辑器需要在保存时，才会正式渲染代码块，无语言高亮时，请尝试保存

```javascript
// 自定义主题变量示例
const theme = {
  accent: '#07C160',      // 微信绿
  fontSize: '16px',
  lineHeight: 1.8,
  textColor: '#333',
};
```

```css
/* .wechat/theme.css */
:root {
  --wechat-accent: #07C160;
  --wechat-font-size: 16px;
  --wechat-line-height: 1.8;
  --wechat-text-color: #333;
  --wechat-code-bg: #f6f8fa;
  --wechat-inline-code-color: #d63384;
  --wechat-blockquote-bg: #f9f9f9;
  --wechat-max-width: 680px;
}
```

```python
# Python 示例
def render_markdown(text: str) -> str:
    """将 Markdown 渲染为微信公众号兼容的 HTML"""
    renderer = WeChatRenderer()
    return renderer.render(text)
```

## 引用块

> 微信公众号编辑器不支持外部 CSS，因此所有样式必须以内联方式注入到每个 HTML 元素上。本扩展自动完成这一过程，确保粘贴后格式完整保留。

> **提示：** 如果图片无法显示，请使用 **✂️ 复制内容** 而非 **📋 复制 HTML**。前者会将本地图片自动转换为 Base64 编码。
